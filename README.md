## filtering-camera

**DEV**: https://localhost:3000

**PRODUCTION**: *TBD*

### Deploy

```shell script
git clone https://github.com/peterdee/filtering-camera
cd ./filtering-camera
nvm use 18
npm ci
```

### Launch

Run locally

```shell script
npm run dev
```

Build application

```shell script
npm run build
```

Serve static files

```shell script
npm run serve
```
